Questão – Sala de Reunião

Descrição

O sistema tem como objetivo controlar o uso de salas de reunião de uma empresa,
permitindo o agendamento, consulta e realocação de reuniões.

Cada reunião possui informações como data, horário, sala, responsável e assunto.

O sistema também permite verificar quais salas estão disponíveis em determinado horário.
 Solução

Foram identificadas as seguintes classes:

- Sala
- Funcionario
- Reuniao
- Agenda

A classe Sala armazena informações sobre as salas disponíveis.

A classe Funcionario representa os colaboradores que utilizam as salas.

A classe Reuniao controla os agendamentos.

A classe Agenda gerencia todas as reuniões.

 Funcionalidades

- Agendamento de reuniões
- Realocação de reuniões
- Consulta por data
- Verificação de salas livres
- Controle de conflitos de horário

Considerações Finais

O sistema facilita o controle e organização das salas de reunião,
substituindo o uso de planilhas manuais por uma solução estruturada
e eficiente.
