Cenário do Projeto
Este projeto simula o controle financeiro pessoal realizado por Vera em uma planilha. O foco é a organização detalhada de saídas financeiras e a geração de inteligência sobre o consumo mensal, permitindo identificar para onde o dinheiro está indo e quais métodos de pagamento são mais utilizados.


Resolução
A estrutura foi pensada para facilitar a análise de dados . No final de cada ciclo mensal, o sistema processa a lista de entradas e gera uma tabela resumo que cruza "O QUE" foi comprado com "COMO" foi pago, oferecendo a Vera um controle total sobre seu orçamento doméstico.
