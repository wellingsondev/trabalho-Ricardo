Cenário
Adriano possui uma vasta coleção de CD's e enfrenta dificuldades para lembrar de todos os itens do seu acervo. O objetivo deste projeto é criar um sistema de catálogo digital para ser utilizado em seu Palm-top, permitindo o registro e a consulta rápida dos álbuns que ele possui.

Resposta da Questão
A solução consiste na modelagem da classe `CD`, que encapsula os atributos básicos identificados no cenário.
