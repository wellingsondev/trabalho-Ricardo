Cenário
Sistema de automação para a padaria do Seu Joaquim. O projeto visa substituir o controle manual por uma comanda eletrônica numerada que acompanha o cliente desde o consumo até o pagamento no caixa.

Resposta da Questão
A solução proposta identifica as classes Comanda, Produto. A relação é estruturada de forma que a Comanda possui uma lista de Produtos. O valor total é um atributo derivado, calculado pelo método finalizarCompra().
