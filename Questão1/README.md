 Descrição

O sistema tem como objetivo controlar os gastos mensais de energia elétrica,
permitindo o cadastro das informações de consumo de cada mês.

Cada conta de luz armazena dados como:

- Data da leitura
- Número da leitura
- Quantidade de KW gasto
- Valor a pagar
- Data de pagamento
- Média de consumo

Além disso, o sistema permite realizar consultas para identificar:

- O mês de menor consumo
- O mês de maior consumo

Funcionamento do Sistema

O sistema permite cadastrar múltiplas contas de luz e realizar análises
sobre o consumo de energia.

Os métodos principais permitem:

- Identificar o menor consumo registrado
- Identificar o maior consumo registrado

Essas informações ajudam no controle financeiro e no acompanhamento do uso de energia.

Tecnologias Utilizadas

- Python
- Streamlit
- GitHub
Considerações Finais

O sistema desenvolvido permite uma visão clara do consumo de energia ao longo
dos meses, facilitando a tomada de decisão e o controle de gastos.

A modelagem foi baseada nos conceitos de orientação a objetos, garantindo
organização, reutilização e facilidade de manutenção.
